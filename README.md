# HTML・CSS 練習リポジトリ

このリポジトリは、HTMLとCSSの学習用に作成したコードをまとめたものです。  
主にWebデザインやフロントエンド開発の基礎を練習しています。

## 📁 内容
- `index.html`：メインページ
- `style.css`：レイアウトとデザイン
- `images/`：使用している画像

## 🧠 学習ポイント
- FlexboxやGridのレイアウト練習
- transition / animation の基本
- レスポンシブデザインの実装
